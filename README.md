# Python Based Ping Pong Game


A simple 2-player Ping Pong (Pong-style) game built with Python and the Turtle graphics library.

Two vertical paddles sit on the left and right sides of the screen. A ball moves across the playfield, bouncing off paddles and the top/bottom walls. Players control their paddle to deflect the ball; if the ball passes a paddle, the opposing player scores a point.

Features

Two-player local gameplay (left player vs right player)

Smooth paddle movement (up / down)

Ball physics: bounce off paddles and top/bottom walls

Score tracking

Adjustable speed and difficulty parameters (move distances, ball speed, etc.)

Controls

Left player: a = move up, w = move down

Right player: Up Arrow = move up, Down Arrow = move down
